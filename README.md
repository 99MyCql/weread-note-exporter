# 微信读书笔记导出器

一个Chrome扩展，适用于微信读书网页版，提供更个性化、更完善的笔记导出。

## 界面预览

![example](./doc/example.png)

## 功能特性

- [x] 自动识别和提取当前书籍的所有笔记
- [x] 支持按章节导出
- [x] 支持复制到剪贴板和导出文件
- [x] 支持Markdown自定义格式
- [ ] 状态保存
- [ ] 多级章节

## 安装和使用

### 开发环境

1. 安装依赖：
   ```bash
   npm install
   ```

2. 构建插件：
   ```bash
   npm run build
   ```

3. 在Chrome中加载插件：
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `dist` 文件夹

### 使用说明

1. 在微信读书网页版打开任意书籍
2. 点击插件图标
3. 选择要导出的章节
4. 点击"复制文本"或"导出文件"
